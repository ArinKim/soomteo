# soomteo - 모바일 샘플 앱

간단한 Expo 기반 React Native 샘플 앱입니다. 이 저장소에는 랜딩 페이지, 로그인(테스트 계정), 로그인 이후 연락처 목록 화면이 포함되어 있습니다.

기능 요약

- 랜딩 페이지: 카카오 스타일 또는 기본 전화(아이폰) 테마 선택 가능
- 로그인: 테스트 계정으로 로그인 가능 (id: 0000 / password: 0000)
- 로그인 이후: 연락처(유저가 추가하거나 미리 등록된 AI 친구 목록) — 각 친구에 대해 전화 또는 채팅 시작 버튼 제공

빠른 시작

1. Node.js와 npm이 설치되어 있는지 확인하세요.
2. 프로젝트 루트에서 의존성을 설치합니다:

```bash
npm install
```

3. Expo를 사용해 실행합니다:

```bash
npm start
# 또는
npx expo start
```

4. Expo Go로 모바일에서 열거나 iOS/Android 시뮬레이터에서 실행하세요.

참고

- 이 프로젝트는 테스트/데모용 간소화된 인증을 사용합니다. 실제 인증이나 백엔드는 포함되어 있지 않습니다.
- 네트워크나 개발 환경에 따라 `npm install`이 필요합니다.
